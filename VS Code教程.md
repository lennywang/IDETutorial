VS Code Emmet



### 非常全的

| 快捷键                 | 功能     | 说明   |
| ------------------- | ------ | ---- |
| Ctrl+Q              | 历史     |      |
| Ctrl+ Shift+ p / F1 | 打开命令面板 |      |



​	

### 基础编辑

| 快捷键                 | 功能       |
| ------------------- | -------- |
| Ctrl + Shift + K    | 删除行      |
| Tab                 | 向右缩进     |
| Shift+Tab           | 向左缩进     |
| Alt+ Shift+  F      | 格式化代码    |
| Alt + Shift + ↑ / ↓ | 向上/向下复制行 |





### 搜索和替换

| 快捷键      | 功能            | 说明   |
| -------- | ------------- | ---- |
| Ctrl + D | 将选择添加至下一个查找匹配 |      |





编辑器管理 

| 快捷键      | 功能    | 说明   |
| -------- | ----- | ---- |
| Ctrl +  | 拆分编辑器 |      |





### 显示

| 快捷键              | 功能            | 说明        |
| ---------------- | ------------- | --------- |
| ctrl + shift + v | 切换Markdown 预览 |           |
| alt + shift + 0  | 切换编辑器布局       | 前提：编辑器已拆分 |



### 集成终端
