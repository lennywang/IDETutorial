| ------------------- | ------ | ---- |
| Ctrl+Q              | 历史     |      |

### 基础编辑

| 快捷键                 | 功能         |
| ------------------- | ---------- |
| Ctrl + Shift + K    | 删除行        |
| Tab                 | 向右缩进       |
| Shift+Tab           | 向左缩进       |
| Alt+ Shift+  F      | 格式化代码      |
| Alt + Shift + ↑ / ↓ | 向上/向下复制行   |
| Ctrl+K Ctrl+0       | 折叠所有区域代码   |
| Ctrl+K Ctrl+J       | 展开所有折叠区域代码 |
| Ctrl+X              | 剪切行        |



### 搜索和替换
