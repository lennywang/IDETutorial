**查看断点处的某个对象的值**

Evaluate Expression (Ctrl +U )

**在调试代码的时候，实时的修改，运行状态的代码变量的值**

Watches

**设置运行时的参数**

Run → Edit Configurations

**keymap**

File → Settings |  keymap

**idea分屏**

File | Settings | 搜索keymap | 搜索Split Vertically （水平分屏）或 Split Horizontally（垂直分屏）

| 右键点击split vertically | 设置水平分屏快捷键
