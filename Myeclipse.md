Myeclipse 常用快捷键

## 快捷键

| 快捷键              | 功能                           | 说明   |
| ---------------- | :--------------------------- | :--- |
| Ctrl+Alt+H       | 选中方法名，查看所有调用此方法的位置           |      |
| ctrl+A->ctrl+i   | 对齐代码                         |      |
| syso -> alt+/    | System.out.println();        |      |
| syst -> alt+/    | System.out.println("当前方法名"); |      |
| F4               | 查看类的结构图                      |      |
| Ctrl+D           | 删除光标所在行                      |      |
| Ctrl+shift+r     | 查找文件                         |      |
| Alt+Enter        | 自动导入包                        |      |
| alt + shift + A  | 选择列                          |      |
| ctrl + alt + ↑/↓ | 复制的行插入在当前行上/下方               |      |
| alt + ↑/↓        | 快速向上/下移动                     |      |
